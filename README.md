## Purpose:

This repository is maintained solely to provide builds of Cine for the Fedora ecosystem.

All core development, feature requests, and upstream bug reports should be directed to the original [project](https://github.com/diegopvlk/Cine)

## Installation:

### Enable the COPR repository
```bash
sudo dnf copr enable nyx004/cine
```
### Install the package
```bash
sudo dnf install cine
```
